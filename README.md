# ReBill

An invoice-verified marketplace that enables users to resell accidental purchases at the original purchase price.

## Problem

Many users accidentally buy products they cannot return. Existing marketplaces encourage bargaining and profit-based resale.

ReBill allows users to sell unused products at the exact invoice price, creating a trustworthy ecosystem.

## Features

- User Authentication
- Invoice Upload
- OCR Invoice Extraction
- Invoice Verification
- Product Listing Creation
- Search and Filtering
- Buyer-Seller Marketplace
- Admin Verification Dashboard
